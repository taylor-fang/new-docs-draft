# API reference

The General Translation API allows you to: upload source content and translated files, download translations, queue translation jobs, manage branches and tags, and read project and job status.

Most SDK and CLI workflows call these endpoints for you. However, the API allows you to call these endpoints directly if you need:

- Custom automation
- Endpoint schemas
- An interactive request playground
- Machine-readable `openapi.yaml` spec

Please refer to the [OpenAPI reference](/docs/platform/openapi/get-started) section.
